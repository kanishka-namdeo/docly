# Superpowers - Design Specs and Implementation Plans

## Purpose

Design specifications and detailed implementation plans for major features and architectural decisions.

## Ownership

This AGENTS.md covers the `docs/superpowers/` directory:
- `specs/` — Design specifications and architectural documents
- `plans/` — Detailed implementation plans and roadmaps

## Local Contracts

### Specs Directory
- `2026-07-10-local-rag-assistant-design.md` — Initial design specification for the local RAG assistant
- Design documents should cover:
  - Problem statement and goals
  - Architecture decisions
  - Component interactions
  - Data models and APIs
  - Trade-offs and alternatives considered

### Plans Directory
- `2026-07-10-local-rag-assistant.md` — Implementation plan for the initial RAG assistant
- Implementation plans should include:
  - Phased approach with clear milestones
  - Task breakdown and dependencies
  - Technical details for each phase
  - Acceptance criteria

### Document Naming Convention
- Format: `YYYY-MM-DD-<feature-name>.md`
- Date reflects creation or major revision date
- Feature name is lowercase with hyphens

## Work Guidance

### Writing Design Specs
1. Start with problem statement and goals
2. Document architecture decisions with rationale
3. Include diagrams for complex interactions
4. Specify APIs, data models, and interfaces
5. Discuss trade-offs and alternatives
6. Define success criteria

### Writing Implementation Plans
1. Break down into phases with clear deliverables
2. Estimate complexity and dependencies
3. Include technical implementation details
4. Define acceptance criteria for each phase
5. Identify risks and mitigation strategies

### Maintaining Documents
- Update when design decisions change
- Archive outdated plans after implementation
- Link to related specs and plans
- Keep documents focused and actionable

## Verification

- Review specs for completeness and clarity
- Verify plans have clear phases and acceptance criteria
- Ensure documents are up-to-date with current implementation
- Check that naming conventions are followed
